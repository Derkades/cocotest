# NutCheck

Expanded test framework for CiviC compiler

## Installation

Run in project root:
```
git submodule add https://github.com/Derkades/nutcheck.git
git add nutcheck
git commit
```

## Usage
```
./nutcheck/test.sh
```

## Updating
```
git submodule update --remote nutcheck
git add nutcheck
git commit
```
