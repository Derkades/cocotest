# Cocotest

Expanded test framework for CiviC compiler

## Installation

Run in project root:
```
git submodule add https://github.com/Derkades/cocotest.git
git add cocotest
git commit
```

## Usage
```
./cocotest/test.sh
```

## Updating
```
git submodule update --remote cocotest
git add cocotest
git commit
```
